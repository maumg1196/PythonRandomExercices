La siguiente matriz (o lista con listas anidadas) debe cumplir la siguiente condición:

En cada fila el cuarto elemento siempre deber ser el resultado de sumar los 3 primeros

Tienes que ser capaz de modificar las sumas incorrectas con el metodo de slicing

matriz = [[1, 1, 1, 3],
		  [2, 2, 2, 7],
		  [3, 3, 3, 9],
		  [4, 4, 4, 13]
]

Pista: La función llamada sum() devuelve una suma total de los elementos de una lista
